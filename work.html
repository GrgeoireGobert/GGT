My work

2020 Mikros Animation - Cage Deformer
I implemented a cage-based Maya deformer. This deformer allows the use of different state of the art coordinate systems on different parts of the same mesh : Green Coordinates, Mean Value Coordinates, Harmonic Coordinates and Maximum Entropy Coordinates. The plugin is paralellized and allows real-time deformation on basic configurations.
Work based on the following publications :

  • Mean Value Coordinates for Closed Triangular Meshes [Ju et al., 2005] : PDF
  • Mean value coordinates for quad cages in 3D [Thiery et al., 2018] : PDF
  • Harmonic Coordinates for Character Articulation [Joshi et al., 2007] : PDF
  • Maximum Entropy Coordinates for Arbitrary Polytopes [Hormann et al., 2008] : PDF
  • Green Coordinates [Lipman et al., 2008] : PDF
  • *Cages: A multi-level, multi-cage based system for mesh deformation [Gonzalez Garcia et al., 2013] : PDF

2020 Mikros Animation - Slide Deformer
I implemented a mesh sliding Maya deformer. This deformer requires the computation of geodesic distance on a mesh and preserves the orignal shape of the mesh while sliding (no volume loss/creation). In addition tp this, I Implemented my own technique to easily design mesh wrinkles onto this sliding. The plugin is paralellized and allows real-time deformation on basic configurations.
Work based on the following publications :

  • Sliding Deformation: Shape Preserving Per-Vertex Displacement [Pinskiy, 2010] : PDF
  • The Heat Method for Distance Computation [Crane et al., 2017] : PDF
